## 为什么要做这个调研？
我们正在筹备搭建一个**全球 GPU 算力租赁 + 大模型 API 平台**，计划支持 A100、H100、RTX 4090/5090、H200、RTX 6000 Pro 等主流显卡。

在正式开发前，我们希望先收集真实用户的需求，你的反馈将直接影响我们的产品方向。

---

## 如何反馈你的需求？

### 方式一：填写外部问卷（推荐）
👉 **[打开调研问卷](https://tally.so/r/your-survey-link)**  
无需登录，国内外均可正常访问。

### 方式二：在 GitHub Issues 留言
请新建一个 Issue，并复制以下模板填写：
```markdown
### 我的 GPU 与大模型需求
- **国家/地区**：
- **最需要的 GPU 型号**：
- **单任务所需显存**：
- **使用场景**：训练 / 推理 / 两者都需要
- **愿意为 RTX 4090 支付的时薪（美元）**：
- **是否需要大模型 API 服务？**：是 / 否
- **是否有闲置 GPU 愿意接入平台**（如有，请填写数量和型号）：
```
