## 📋 问卷内容预览

1. **使用场景** – 训练 / 推理 / 科学计算 / AI 绘图  
2. **GPU 型号** – A100、H100、H200、4090、5090、6000 Pro 等  
3. **单任务显存需求** – <16GB 到 >128GB  
4. **单卡还是多卡** – 1 卡、2-4 卡、5-8 卡、>8 卡；是否需要 NVLink/NVSwitch  
5. **使用方式** – 按秒/小时租用（SSH/Jupyter） / 长租包月/包年  
6. **价格敏感度** – 你愿意为 RTX 4090 每小时支付多少美元  
7. **大模型 API 月预算**  
8. **所在国家/地区**  
9. **是否有闲置 GPU 愿意接入平台赚收益**  
10. **其他建议**
