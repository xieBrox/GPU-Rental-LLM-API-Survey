## Why this survey?
We are exploring the idea of building a **global GPU compute server rental + LLM API platform** that provides dedicated GPU servers with models like A100, H100, RTX 4090, RTX 5090, H200, RTX 6000 Pro, etc.

Before we build anything, we want to hear from real users like you. Your feedback will directly shape our product roadmap.

---

## 🔗 Survey Link
👉 **[Fill out the survey on Tally](https://tally.so/r/yPo7Pp)**  
(No login required, accessible worldwide)

---

## 📋 What we ask (preview)

1. **Use case** – training / inference / scientific computing / AI generation  
2. **GPU models** – A100, H100, H200, 4090, 5090, 6000 Pro, etc.  
3. **VRAM per task** – <16GB to >128GB  
4. **Single or multi-GPU** – 1, 2-4, 5-8, >8 cards; need NVLink/NVSwitch?  
5. **Access model** – pay-as-you-go (SSH/Jupyter) / long-term subscription  
6. **Price sensitivity** – hourly rate you'd pay for RTX 4090 (USD)  
7. **LLM API monthly budget**  
8. **Your country/region**  
9. **Spare GPUs to share** (earn rewards)  
10. **Any other suggestions**
