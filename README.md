## Why this survey?
We are exploring the idea of building a **global GPU rental + LLM API platform** that supports GPUs like A100, H100, RTX 4090, RTX 5090, H200, RTX 6000 Pro, etc.

Before we build anything, we want to hear from real users like you. Your feedback will directly shape our product roadmap.

---

## How to share your needs?

### Option 1: Fill out the external form (Recommended)
👉 **[Open the Survey](https://tally.so/r/your-survey-link)**  
No login required, works worldwide.

### Option 2: Submit a GitHub Issue
Use the template below to create a new Issue:
```markdown
### My GPU & LLM Needs
- **Country/Region**: 
- **GPU models I need most**: 
- **Typical VRAM required per task**: 
- **Use case**: Training / Inference / Both
- **Willing to pay (USD/hour) for e.g. RTX 4090**: 
- **Do I need LLM API access?**: Yes / No
- **I have spare GPUs to share** (if yes, please specify quantity and models): 
```
